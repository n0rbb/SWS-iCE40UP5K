# SENSOR DE ONDAS DE ESPÍN - CONTADORES CON iCE40UP5K

En este repositorio se encuentran los ficheros de VHDL, así como los ficheros de cotas empleados para sintetizar el _hardware_ empleado para los contadores para sensores de ondas de espín. 
El repositorio incluye además los dos IPCores de Lattice empleados en el proyecto (EBR - RAM y PLL - FQCLK). 

---

## 📝 Notas

Los ficheros de simulación se encuentran en la misma carpeta que las fuentes de VHDL y los ficheros de cotas (.sdc y .pdc): source/impl_1
